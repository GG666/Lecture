## Time Complexity

在資訊工程這項領域，有個非常重要的部分，叫做時間複雜度，時間複雜度就是分析我們的程式執行時間的方式，主要有O(upper bound), Theta(tight bound), Omega(lower bound),這三種，舉個簡單的例子
```C++
#include<iostream>
int main(){
    int n;
    for(int i=0;i<n;i++){
        //do something                      
    }
}
```
                          
上面的程式的時間複雜度可以寫成O(n),O(n^2),Theta(n),Omega(1)等等
